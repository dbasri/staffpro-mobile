# **App Name**: StaffPro Mobile

## Core Features:

- User Authentication: Secure user authentication using passkeys, MFA verification codes, and/or username/password.
- Session Management: Secure session management to maintain user authentication state.
- Web Content Wrapper: Display web page content within a wrapper that hides browser menus and controls.
- Content Loading and Display: Fetch and display web content within the controlled environment.

## Style Guidelines:

- Primary color: Deep Indigo (#F7941D) to convey security and trust.
- Background color: Light Gray (#818286) for a clean, distraction-free background.
- Accent color: Teal (#009688) to highlight important interactive elements such as call-to-action buttons.
- Body and headline font: 'Inter' sans-serif for a clean and modern look. It is suitable for both headlines and body text, making it great for applications with small amounts of text.
- Use simple, clear icons for navigation and authentication states.
- A clean and focused layout that emphasizes content security and ease of use.
- Subtle transitions during authentication and content loading to improve user experience.
